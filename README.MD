### Prerequisites

Make sure you have **Java JDK** installed (version 8 or higher).

Check Java version:

```bash
java -version
```

### Run from Source

1. Clone the repository:

   ```bash
   git clone https://github.com/Adiraj001/Java-Calculator.git
   ```
2. Navigate into the project folder:

   ```bash
   cd Java-Calculator
   ```
3. Compile the source:

   ```bash
   javac src/*.java
   ```
4. Run the calculator:

   ```bash
   java -cp src Calculator
   ```
